Toy Robot Test
1.Use ""PLACE X,Y,Direction"" command First!
2.Use MOVE command to one step towards the current location and direction
3.Use RIGHT command to rotate one step Clockwise
4.Use LEFT command to rotate one step Counter-Clockwise

not required in the Robot Test criteria but I think useful for the users :)

5.OPTIONAL TEST command -->Use TEST command to Test the Different Robot Commands and it's limitations
6.Commands are handled to be not case sensitive
7.Invalid Place command values will be ignored, the robot will not move and remain in the last valid position.
8.type EXIT or exit command to close the window
